This project demonstrates how **Convolutional Neural Networks (CNN)** can learn patterns from handwritten digits and classify them accurately.

The system analyzes pixel patterns, edges, and shapes to predict digits **0–9**.

---

# 🧠 AI Model Overview

The model uses **Deep Learning CNN architecture** to perform image classification on the MNIST dataset.

# 📊 MNIST Dataset

The MNIST dataset is a popular benchmark dataset in machine learning.

| Property        | Value          |
| --------------- | -------------- |
| Dataset Size    | 70,000 images  |
| Training Images | 60,000         |
| Testing Images  | 10,000         |
| Image Size      | 28 × 28 pixels |
| Classes         | 10 digits      |
